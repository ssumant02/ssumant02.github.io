Remote Control Vehicle

Overview

A three-speed remote control vehicle featuring an automatic transmission (low, high, reverse) on a pickup-style chassis. The project explores mechanical design, linkage geometry, drivetrain efficiency, and closed-loop control for smooth gear changes under varying loads.

Gallery

What

This project involved creating a remote controlled vehicle equipped with a three gear transmission: Low, High, and Reserve. The vehicle features automatic gear shifting that is controlled through the remote, allowing smooth transitions between gears without manual intervention.

How

CAD for gearbox and linkages; fabricated components; tuned servo actuation and throttle ramps; implemented controller logic for shift timing; logged runs for efficiency and speed tests.

Results

Achieved consistent shifts with minimal slip; validated torque handling across ratios; improved lap consistency and top speed; identified next steps for weight reduction and gearing optimization.
